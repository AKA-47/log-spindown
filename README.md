# log-spindown

Collection of scripts and cron jobs to log HDD spin-down.

Dependencies: hdparm

Outputs a log file: /var/log/spindown.log

--------------------------------
to. 17. nov. 19:10:01 +0100 2016

/dev/sdb:
 drive state is:  standby

/dev/sdc:
 drive state is:  standby
--------------------------------
